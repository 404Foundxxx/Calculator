# Calculadora

<img src="./calculator.png" alt="Descripción" width="250">

Proyecto simple de una calculadora web.

## Archivos

- `index.html` - Interfaz HTML de la calculadora.
- `main.js` - Lógica JavaScript.
- `styles.css` - Estilos.
- `calculator.png` - Imagen de la calculadora mostrada arriba.

## Uso

1. Abre `index.html` en tu navegador.
2. Usa la interfaz para realizar operaciones básicas.
